# 视觉风格与预设方案参考 (Style Presets Reference)

为 Frontend Slides 幻灯片量身定制的高级视觉风格大观。这里的每一个预设均汲取了真实世界中的高级设计灵感与排版规律，坚决抵制粗劣且毫无美感的“AI 廉价画风”。**所有装饰均采用抽象 CSS 图形——完全禁用具象化的插画图库。**

**自适应视口 CSS 声明：** 任何主题都必须优先挂载强制性底层样式 [viewport-base.css](../assets/viewport-base.css)，它是每一次构建都不可或缺的渲染底座。

## 主基调：暗黑系统 (Dark Themes)

### 1. 强力讯号 (Bold Signal)

**情绪氛围 (Vibe):** 自信、硬核、极简现代、极具视觉冲击力
**排版布局:** 采用纯黑色渐变背景，以极高饱和度的点缀撞色卡片作为中心锚点。左上角数字、右上角导航、左下角标题。

**排版字体:**
- 巨幕标题 (Display): `Archivo Black` (900字重)
- 正文 (Body): `Space Grotesk` (400/500字重)

**色板配方:**
```css
:root {
    --bg-primary: #1a1a1a;
    --bg-gradient: linear-gradient(135deg, #1a1a1a 0%, #2d2d2d 50%, #1a1a1a 100%);
    --card-bg: #FF5722;
    --text-primary: #ffffff;
    --text-on-card: #1a1a1a;
}
```

**灵魂元素 (Signature):**
- 作为绝对视觉焦点的亮色高弹卡片（橙、珊瑚红或极度吸睛的强调色）
- 超大号的纪元式段落编号 (如：01, 02)
- 通过透明度控制激活状态的面包屑微导航
- 基于绝对网格线的严丝合缝对齐体系

## 🚫 绝对抵制垃圾 AI 塑料味审美 (DO NOT USE)

哪怕不指定主题，Agent 系统也**严格禁止**做出下列产生廉价大众模板感的行为：
- **烂大街字体**：在核心标题大篇幅使用未经设计的系统白名单字比如 Arial、Roboto 或者干瘪的 Inter（如果没调教字间距）。
- **廉价色彩**：滥用未经调和的纯白（#fff）加上土得掉渣的俗不可耐紫罗兰渐变，亦或者是名为靛蓝（Indigo #6366f1）的纯工业色。
- **排版灾难**：无脑居中所有东西、搞一些自以为是的巨大圆润水晶玻璃态（Glassmorphism）、加上浓墨重彩却瞎比投射的厚重下拉阴影（Drop Shadows）。
- **最大的死罪**：使用那些虚假、浮面的 3D 画廊插图或没内涵的矢量插画！如果需要插图，老老实实写点有深度的字或用抽象几何图形。

## ⚠️ CSS 高级坑点警务室 (CSS Gotchas)

### 当你需要抵消一个流体 CSS 函数时

**❌ 大错特错 —— 浏览器会偷偷完全忽略它（且控制台没有任何报错）：**
```css
right: -clamp(28px, 3.5vw, 44px);   /* 这行代码在浏览器眼里是个废气 */
margin-left: -min(10vw, 100px);     /* 这行代码也会被直接当成耳旁风 */
```

**✅ 绝对方程式 —— 必须在外面包裹上计算器罩 `calc()`：**
```css
right: calc(-1 * clamp(28px, 3.5vw, 44px));  /* 绝对管用！完美反转流体极限 */
margin-left: calc(-1 * min(10vw, 100px));     /* 绝对管用！ */
```
**原因破译：** W3C 原生 CSS 编译器根本不承认在像 `clamp()` 这样的高级液态响应函数前面擅自加上 `-` 号做成负数反演。遇到它，浏览器为了不宕机就选择静默处理，权当没看见。永远记住：**你要让动态数值反转，必须用 `calc(-1 * [函数])` 去强行负相乘放大！**
